# windows命令 #笔记

> 数据来源：[知乎windows命令介绍](https://zhuanlan.zhihu.com/p/426749071)

## 1. 位置变动

   | 键位       | 作用                        |
   | ---------- | --------------------------- |
   | `cd`      | 目录移动(绝对`str`/相对`.`) |
   | `dir/tree` | **查看目录/目录树**         |

## 2. 文件处理

   | 键位       | 作用              |
   | ---------- | ----------------- |
   | `md/mkdir` | 创建目录          |
   | `rd`      | 移除目录          |
   | `type`     | **查看/创建文件** |
   | `echo`     | 创建非空文件      |
   | `del`      | 删除文件          |

## 3. 输入

   | 键位  | 作用 |
   | ----- | ---- |
   | `tab` | 补全 |
   | `cls` | 清屏 |
